# PortfolioHUB


Portfólio acadêmico e profissional desenvolvido na disciplina de Bootcamp do
    primeiro período de Ciência da Computação.


## Objetivo
Centralizar projetos, experiências, habilidades e links profissionais, utilizando
    GitHub para versionamento e GitHub Pages para publicação.


## Tecnologias
- HTML5
- CSS3
- Git e GitHub
- GitHub Pages
- Google Workspace
- Google Gemini como ferramenta de apoio


## Estrutura
- `index.html`: conteúdo e estrutura da página;
- `style.css`: identidade visual e responsividade;
- `images/`: imagens utilizadas no portfólio;
- `.gitignore`: arquivos que não devem ser versionados.


## Acesso
- Site: https://caduavelarr.github.io/PortfolioHUB/
- Google Sites: https://sites.google.com/view/caduavelarr/inicio


## Segurança
Nenhuma senha, chave de API, token ou dado sensível deve ser enviado ao
    repositório. As alterações na branch principal devem ser revisadas antes da
    publicação.
